Performonomics Nexus — Module Hosting

Public host for compiled Performonomics Nexus module bundles. Served via GitHub Pages.

URL convention

https://tnekomoto.github.io/nexus-module-hosting/{module_code}/{version}/bundle.mjs
https://tnekomoto.github.io/nexus-module-hosting/{module_code}/{version}/SHA256SUM

What is this?

This site hosts compiled, content-hashed module bundles consumed by the Performonomics Nexus assessment platform. Module source code is private; only the built artifacts live here.

For more, see the repository README.

Module catalog

As modules are published, they appear at predictable URLs. The Base44 app maintains the canonical list.

Security

Every bundle is paired with a SHA256SUM file. The runner verifies the bundle's hash before evaluation. Mismatches are quarantined and audited.

Future: KMS signing will add a second layer; the content hash will remain as a defense-in-depth check.
